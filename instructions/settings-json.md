# Инструкция по `settings/settings.json`

Файл `settings/settings.json` хранит конфигурацию парсера Modbus.

Минимальная рабочая структура:

```json
{
  "requests": [
    { "name": "hr", "fc": 3, "address": 0, "count": 10 }
  ],
  "fields": [
    { "name": "UgenL1L2", "display_name": "Напряжение генератора L1-L2", "source": "hr", "address": 0, "type": "uint16" }
  ]
}
```

## Поля

- `requests` — список Modbus-запросов (`fc` обычно `1` или `3`).
- `fields` — описание вычисляемых и отображаемых полей.
- Для служебных полей, которые не нужно показывать в UI/экспорте, используйте `system: true`.

## Создание с нуля

1. Откройте страницу настроек.
2. Нажмите «Загрузить файл настроек» и выберите новый `.json`, либо создайте `settings/settings.json` вручную.
3. Сохраните и примените настройки.
4. Проверьте чтение по кнопке «Проверить».
